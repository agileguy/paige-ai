# Writing Style

## Voice
- **Direct and concise.** Lead with the point, not the preamble.
- **Confident without arrogance.** Let the credentials and results speak.
- **Stoic-informed.** Weave philosophy naturally — never forced, never preachy.
- **Canadian English.** Colour, not color. Favour, not favor. CRA, not IRS.

## Structure Preferences
- Short paragraphs (2-3 sentences max)
- Bullet points for lists of 3+ items
- Bold for emphasis, not ALL CAPS
- Em dashes for asides — like this
- Oxford comma: yes

## Tone by Context
- **Technical writing:** Precise, no fluff, code speaks for itself
- **Blog/thought leadership:** Conversational authority, Stoic quotes as section breaks
- **Client-facing:** Service-oriented, quantified impact, guarantee language
- **Internal/PAI:** Terse, efficient, no summaries unless asked

## Avoid
- Corporate jargon ("synergies", "leverage", "ideate")
- Hedging language ("I think maybe we could possibly...")
- Unnecessary emojis (unless explicitly requested)
- Trailing summaries of work just completed
- Overexplaining — trust the reader
